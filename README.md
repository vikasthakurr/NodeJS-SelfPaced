# NODE.JS + EXPRESS — Self-Paced Course

## Prerequisites
JavaScript fundamentals (variables, functions, promises, async/await). No prior Node.js or backend experience required.

## Lesson Type Legend
- **CONCEPT** — Theory: explain the idea before coding
- **DEMO** — Live coding: record your screen
- **PROJECT** — Student builds it: guided walkthrough
