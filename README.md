# Node.js + Express — Self-Paced Course

Hey, welcome to the course! 👋

I'm Vikas, and I built this course to teach you backend development with Node.js and Express from scratch. No fluff, no filler — just real code, real patterns, and real projects you can put on your portfolio.

By the end of this course you'll be able to design, build, and deploy production-ready REST APIs with authentication, databases, caching, real-time features, and proper documentation. Whether you're a frontend dev looking to go full-stack or a beginner stepping into backend for the first time — this is for you.

Go at your own pace. Each section builds on the last, so start from the top and work your way through. You'll be shipping APIs before you know it.

## Prerequisites

- JavaScript fundamentals (variables, functions, promises, async/await)
- No prior Node.js or backend experience required

## Lesson Types

| Tag | Meaning |
|-----|---------|
| CONCEPT | Theory — understand the idea before coding |
| DEMO | Live coding — follow along on screen |
| PROJECT | Guided walkthrough — you build it |

## Course Outline

### Core

| # | Section | Focus |
|---|---------|-------|
| 01 | Introduction & Setup | Node.js basics, npm, first script |
| 02 | Node.js Core Modules | fs, path, events, raw HTTP server |
| 03 | Express Fundamentals | Routing, middleware, request/response |
| 04 | Building a REST API | CRUD, validation, error handling |
| 05 | Databases — MongoDB + Mongoose | Schemas, queries, relationships |
| 06 | Authentication & Security | JWT, bcrypt, role-based access |
| 07 | File Uploads & Environment Config | Multer, dotenv, Nodemailer |
| 08 | Testing & Code Quality | Jest, Supertest, ESLint |
| 09 | Deployment & Production | Render/Railway, PM2, logging |

### Advanced

| # | Section | Focus |
|---|---------|-------|
| 10 | WebSockets & Real-time | Socket.io, rooms, live chat |
| 11 | Redis & Caching | ioredis, cache patterns, BullMQ |
| 12 | API Documentation with Swagger | OpenAPI, swagger-jsdoc, interactive docs |
| 13 | Capstone Project | Full API combining everything learned |

## Getting Started

```bash
git clone https://github.com/vikasthakurr/NodeJS-SelfPaced.git
cd NodeJS-SelfPaced
```

Each section folder contains its own README with lesson details. Navigate into any lesson subfolder, run `npm install`, and follow along.
